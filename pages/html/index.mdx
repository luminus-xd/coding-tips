# HTML

HTML（HyperText Markup Language）は、ウェブページを作成するための標準マークアップ言語です。  
テキストコンテンツとその構造を記述するために使用され、ウェブページの基盤を形成します。

## HTML Living Standard と W3C

HTML の開発と標準化は、主に二つの組織、W3C（World Wide Web Consortium）と WHATWG（Web Hypertext Application Technology Working Group）によって進められています。

- W3C: ウェブ技術の標準化を目指す国際コンソーシアム。HTML4 や XHTML など、初期の HTML 仕様を策定。
- WHATWG: Apple, Google, Mozilla などのブラウザベンダーが中心となり、HTML の開発を進めるグループ。HTML を「生きた標準（Living Standard）」として継続的に更新。
